skytime handle multiple messages from the server in realtime without closing/reconnecting.

1. run make
2. open **<http://localhost:8000>** and choose an websocket
3. connect to localhost port 8001 via telnet (```telnet localhost 8001```)

for more informations about real-time technologies you can read reference:
[AJAX vs Long-Polling vs SSE vs Websockets vs Comet)] http://stackoverflow.com/a/12855533
